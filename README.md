# Marvellous-Study-Tracker
Marvellous Study Tracker is a Java-based console application designed to help students track their daily study activities efficiently. 
 It serves as a digital study diary that enables better time management and productivity tracking.
 
# OverView:

This project is a Java-based console application designed to help students track their daily study activities efficiently. The system allows users to record the subject they studied, the number of hours spent studying, and a description of the study session.
The project demonstrates the practical implementation of Java programming concepts such as Object-Oriented Programming, Java Collections Framework, and file handling.

# Technologies Used : 

    Language : Java
    Packages & APIs
    java.util.* → Data structures (ArrayList, TreeMap), user input via Scanner.  
    java.time.LocalDate → Auto-captures the current date for study logs.  
    java.io.File handling and CSV export. 
    Java Collections Framework.  
    CSV File Export
    Object-Oriented Programming (OOP)
    
#  This project provides : 

# 1 : Insert Log()
    
          Users can insert new study records with the following details:

          Date (fetched automatically from system)  
          Subject  
          Duration (in hours)  
          Description of the study activity  


# 2 : Display Log() 

          Displays all study records in a structured manner.  
          
          Each entry shows:  
          
              Date  
              Subject  
              Time spent  
              Description  


# 3 : ExportCSV() 

            Allows users to export all study records into a CSV file, which can be opened using Excel or other spreadsheet software.

# 4 : SummaryByDate() 

            Generates a summary showing total study hours grouped by date.

# 5 : SummaryBySubject()

            Generates a summary showing the total hours spent studying each subject.

# 6 : Menu Driven Interface

            User-friendly console menu with numbered options:

            Insert Log  
            Display Log   
            Export to CSV  
            Summary by date  
            Summary by subject  
            Exit  

# Key Features : 

          Add study session details  
          Automatic date recording for each entry  
          Menu-driven console interface  
          Efficient data storage using Java Collections Framework  

# Key Benefits : 

          Helps students identify and analyze study patterns.  
          Improves time management and productivity.  
          Provides structured data for academic tracking through CSV export.
